# Strømly – Landing Page

Dette projekt er en simpel **coming soon landing page** for virksomheden **Strømly**.

Siden er bygget i **Next.js + React + Tailwind CSS** og fungerer som en midlertidig hjemmeside frem mod lancering i 2026.

---

##  Tech Stack

- **Next.js (App Router)**
- **React**
- **TypeScript**
- **Tailwind CSS**

---

##  Projektstruktur

```txt
src/
 ├── app/
 │   ├── page.tsx        → Forside (Landing page)
 │   ├── layout.tsx      → Global layout
 │   └── globals.css     → Global styling
 │
 ├── Public              → Genbrugelige komponenter (senere)
 │
 └── Src/                → Undersider
